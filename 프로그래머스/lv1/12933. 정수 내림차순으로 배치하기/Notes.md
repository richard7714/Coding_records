### 1. string에서 sort 라이브러리 활용
- sort는 string도 정리해준다.
- sort(str.begin(), str.end()) => 오름차순 정리
- sort(str.begin(), str.end(), greater<char> ()) => 내림차순 정리
  
### 2. string to (타 변수 type)
  - stoi() => string to int
  - stoll() => string to long long
  - stof() => string to float
  - stod() => string to double
